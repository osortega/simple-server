# Simple Server

A simple Express.js server that provides a basic HTTP API.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

```bash
npm install
# or
yarn install
```

### Running the Server

```bash
npm start
# or
yarn start
```

The server will start on port 3002.

## API Documentation

### GET /

Returns a greeting message.

**Request:**
```
GET http://localhost:3002/
```

**Response:**
```
Hello world!
```

**Status Code:** 200 OK
